This site presents player statistics derived from official Chivalry 2 server sources. The gathering process is time-consuming and updates are done in batches. Please note that the available data is limited and certain restrictions apply. At present, updates are ~daily targeted at 100,000 top players..

Assistance with the web frontend and the creation of an aesthetically pleasing UI would be appreciated. If you or someone you know possesses the required skills, please reach out.
